                Fediska Injector v5.6
        Modern, Fast and Stable DLL Injector
========================================================

Описание:
Fediska Injector — это современный DLL-инжектор для Windows 7–11,
созданный для простой и быстрой инъекции библиотек в 32/64-бит процессы.
Интерфейс лёгкий, понятный, а лог-инжект показывает результат работы.

--------------------------------------------------------
Возможности:
--------------------------------------------------------
- ManualMap и LoadLibrary инжект
- Выбор процесса через его PID или название процесса
- Чистый, современный интерфейс
- Логи инжекции в реальном времени
- Быстрая загрузка DLL
- Минимальные срабатывания антивирусов
- Полная поддержка Windows 7/8/10/11

--------------------------------------------------------
Как использовать:
--------------------------------------------------------
1. Запустите FediskaInjector.exe
2. Нажмите "Select Process" и выберите нужный процесс
3. Нажмите "Add DLL" и выберите вашу .dll
4. Нажмите кнопку "Inject"
5. Результат появится в окне логов

Системные требования:
--------------------------------------------------------
- Windows 7 / 8 / 10 / 11
- x86 или x64 процесс
- .dll файл для инъекции
